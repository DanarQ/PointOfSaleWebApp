# Point Of Sale Web App

Point Of Sale Web App adalah aplikasi kasir berbasis web untuk swalayan. Aplikasi ini dirancang untuk membantu proses operasional toko, mulai dari pengelolaan produk, kategori, stok barang, transaksi penjualan, hingga laporan dasar.

## Status Pengembangan

Project ini masih dalam tahap pengembangan aktif. Struktur aplikasi, fitur, tampilan antarmuka, dan integrasi backend masih dapat berubah sesuai kebutuhan selama proses pembangunan.

## Rencana Fitur

- Manajemen produk dan kategori.
- Pengelolaan stok barang.
- Proses transaksi kasir.
- Riwayat transaksi penjualan.
- Autentikasi pengguna.
- Laporan operasional toko.
- Dashboard ringkasan untuk pemilik atau admin toko.

## Struktur Project

```text
PointOfSaleWebApp/
+-- backend/   # API, database, autentikasi, dan logika server
+-- frontend/  # Tampilan web aplikasi POS
```

## Catatan

Aplikasi ini belum siap untuk digunakan di lingkungan produksi. Dokumentasi, instruksi instalasi, dan panduan penggunaan akan dilengkapi seiring perkembangan fitur.
